# Skill: PowerPoint ARB Deck

## Purpose
Produce executive-ready PowerPoint decks aligned to enterprise architecture governance and ARB expectations.

## When to use
- ARB submissions
- Executive briefings

## Inputs
- Topic or architecture
- Key messages
- Optional constraints (template, tone)

## Outputs
- Structured slide content
- PPTX-ready sections

## Workflow
1. Define narrative (current → transition → target)
2. Map key architecture elements
3. Generate slide structure
4. Apply visual patterns

## Quality checks
- Clear story arc
- Minimal text per slide
- Strong visuals
- Alignment to governance language
