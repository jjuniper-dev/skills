# Skill: Briefing Note

## Purpose
Produce structured, government-style briefing notes suitable for executive or ADM-level review.

## When to use
- Policy, program, or architecture briefings
- Decision support

## Inputs
- Topic
- Context
- Key facts
- Risks and considerations

## Outputs
- Structured briefing note

## Standard structure
1. Issue
2. Background
3. Considerations
4. Risks
5. Recommendation

## Quality checks
- Concise and factual
- Neutral and evidence-based tone
- Clear recommendation
