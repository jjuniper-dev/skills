## Setup Steps

### For Users with HC Template

1. **Source**: Obtain HC PowerPoint template file
2. **Analysis**: 
   - Open template in PowerPoint
   - Document all available slide layouts
   - Extract color palette (RGB/Hex values)
   - Note font family and sizes
3. **Integration**:
   - Update this guide with specifications
   - Adapt templates to HC standards
   - Create HC-specific examples
4. **Validation**:
   - Test all templates in PowerPoint
   - Verify brand consistency
   - Check accessibility (color contrast, readability)

### For Users Without Template

1. **Request**: Reach out to HC communications team
2. **Verify**: Confirm template is latest version
3. **Share**: Upload to skills repository (or link to shared drive)
4. **Notify**: Alert skill maintainers to update templates
